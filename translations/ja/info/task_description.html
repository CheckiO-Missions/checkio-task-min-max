max(iterable, *[, key]) or min(iterable, *[, key])
max(arg1, arg2, *args[, key]) or min(arg1, arg2, *args[, key])

引数iterableの中の最大値(最小値)、もしくは2つ以上の引数の最大値(最小値)を返却する

引数が一つの場合、それはiterableであるべきです。最大値(最小値)の要素をその中から返します。 引数が2個もしくはそれ以上の場合、引数の中の最大(最小値)を返します。

オプションのキーワード引数keyは、一つの引数を受け取る関数を指定します。 その関数はリストの各要素から比較するキーを抽出するのに利用されます。(例: key=str.lower)

もし複数の最大値(最小値)の要素が見つかったら、関数は最初に見つかったものを返します。

-- Python Documentation (Built-in Functions)

入力: iterableな単一の引数、もしくは2つ以上の引数。(オプション) キーワード引数形式の関数。

出力: "max" 関数の最大値 と "min" 関数の最小値.

例: 

max(3, 2) == 3
min(3, 2) == 2
max([1, 2, 0, 3, 4]) == 4
min("hello") == "e"
max(2.2, 5.6, 5.9, key=int) == 5.6
min([[1, 2], [3, 4], [9, 0]], key=lambda x: x[1]) == [9, 0]

どう使われるか: この問題は、組み込み関数のいくつかについて、より正確なレベルでどのように機能するかを理解するのに役立ちます。

事前条件: 全てのテストケースは正確で、関数は例外を発生させる必要がありません。
