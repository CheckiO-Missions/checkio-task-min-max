Dans cette mission, vous devez écrire votre propre implémentation py3 (mais vous pouvez aussi utiliser py2 pour cela) des fonctions built-in min et max. Il vous est interdit d'utiliser certaines fonctions built-in : import, eval, exec, globals. N'oubliez pas qu'il vous faut implémenter deux fonctions dans votre code !

max(iterable, *[, key]) vagy min(iterable, *[, key])
max(arg1, arg2, *args[, key]) vagy min(arg1, arg2, *args[, key])

Revoie le plus grand (respectivement le plus petit) item d'un itérable ou le plus grand (resp. le plus petit) de deux arguments ou plus.

Si un seul argument est fourni, ce doit être un itérable. Le plus grand (resp. petit) item de l'itérable est renvoyé. Si deux arguments ou plus sont fournis, le plus grand (resp. petit) des arguments est renvoyé.

L'argument keyword-only "key" spécifie une fonction à un paramètre qui est utilisée pour effectuer la comparaison des éléments ( par exemple : key = str.lower ).

Si plusieurs éléments sont maximaux (resp : minimaux), la fonction retourne le premier rencontré.

-- documentation Python (Fonctions built-in

Input: Un argument itérable ou bien plusieurs arguments. Argument keyword optionnel (une fonction).

Output: Le plus grand item pour la fonction "max" et le plus petit item pour la fonction "min".

Exemple: 

max(3, 2) == 3
min(3, 2) == 2
max([1, 2, 0, 3, 4]) == 4
min("hello") == "e"
max(2.2, 5.6, 5.9, key=int) == 5.6
min([[1, 2], [3, 4], [9, 0]], key=lambda x: x[1]) == [9, 0]

Utilisation: Cette mission va vous aider à comprendre plus précisément le fonctionnent de certaines fonctions built-in.

Préconditions: Tous les cas de tests sont corrects et les fonctions ne doivent pas lever d'exceptions.
